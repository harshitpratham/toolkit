NotebookLM
AI research assistant grounded in your documents
Included · no extra cost AI-powered
▼

What to use it for

  • Upload a long report or policy doc — ask it questions, get cited answers
  • Paste in multiple project docs and create a shared team notebook
  • Generate an Audio Overview — a podcast-style summary of any document set
  • Onboard new staff by building a notebook from SOPs and FAQs
  • Research synthesis: upload 10 papers, ask it to compare findings

How to get started

  1. Go to notebooklm.google.com (sign in with your Pratham account)
  2. Click "New Notebook" → add sources: Google Docs, PDFs, URLs, or pasted text
  3. Type a question in the chat — answers are cited with source references
  4. Try "Audio Overview" to generate a 5-min spoken summary
  5. Share the notebook with colleagues for collaborative research
→ notebooklm.google.com

Google AI Studio
Prompt & test Gemini models directly — no code needed
Free with Workspace account AI-powered Power users
▼

What to use it for

  • Test and refine AI prompts before building them into a workflow
  • Upload a PDF or spreadsheet and ask Gemini to analyse or extract data
  • Build a custom system prompt for a repeatable task (e.g. report drafting)
  • Experiment with Gemini's latest models (including 2.5 Pro) for free
  • Generate structured outputs (JSON, tables) from unstructured text

How to get started

  1. Go to aistudio.google.com — sign in with your Pratham Workspace account
  2. Click "New prompt" → choose a model (start with Gemini 2.5 Flash)
  3. Type a system prompt (e.g. "You summarise field reports into 3 bullets") then test it
  4. Upload files via the paperclip — PDFs, images, CSVs all work
  5. Save your prompt and share the link with your team
→ aistudio.google.com

Workspace Studio
Build no-code AI automations across your Workspace apps
Included · rolling out now AI automation
▼

What to use it for

  • Auto-summarise unread emails each morning and send to your Google Chat
  • When someone fills a Google Form → save response to Sheets + notify in Chat
  • After a Meet call → extract action items and draft a follow-up email
  • Label and prioritise incoming emails using Gemini
  • Save email attachments to a Drive folder automatically

How to get started

  1. Go to studio.workspace.google.com (may still be rolling out — check with Harshit)
  2. Browse pre-built templates on the Discover page
  3. Or describe your automation in plain language — Gemini builds the flow for you
  4. Click "Test run" to verify it works, then turn it on
  5. Flows run in the background on any device once active
→ studio.workspace.google.com

Gemini App
AI assistant for research, writing, and thinking
Included AI assistant
▼

What to use it for

  • Brainstorm frameworks for a proposal or strategy doc
  • Draft a first version of a report, blog, or email
  • Ask it to simplify a complex document into plain language
  • Create a Gem — a reusable AI persona (e.g. "Field report editor")
  • Use Deep Research mode for multi-source topic synthesis

How to get started

  1. Go to gemini.google.com — sign in with your Pratham account
  2. Type a task (e.g. "Help me write an agenda for a program review meeting")
  3. Upload a file or paste text as additional context
  4. Deep Research (fast mode) is available at no extra cost — use it freely
  5. Note: Gemini Pro (advanced model) has a daily usage cap; stick to the standard model for everyday tasks
→ gemini.google.com

Google Vids
AI-assisted video creation for internal comms and training
Included · AI features active until May 2026 AI video
▼

What to use it for

  • Create a short training video without recording yourself — use an AI avatar
  • Turn a Docs-based SOP into a narrated walkthrough video
  • Build a program overview video for donors or partners
  • Record a quick explainer with screen + voiceover for remote teams
  • Export directly to Drive and share via link

How to get started

  1. Go to vids.google.com or open it from the Google apps grid
  2. Click "New video" → describe what you want to make
  3. Choose a template or let Gemini generate a storyboard for you
  4. Add narration: record your voice, use text-to-speech, or pick an AI avatar
  5. Preview and export — saves directly to your Drive
→ vids.google.com

AppSheet
Build custom apps and automations from Google Sheets — no code
Basic tier included No-code builder
▼

What to use it for

  • Build a mobile data-entry app for field teams (works offline)
  • Turn a Sheets attendance tracker into a phone-friendly form
  • Create a simple task manager or approval workflow for your team
  • Build a beneficiary registration app without writing a line of code
  • Auto-send notifications when a row in Sheets is updated

How to get started

  1. Open a Google Sheet with your data
  2. Go to Extensions → AppSheet → Create an app
  3. AppSheet auto-generates a mobile app from your Sheet structure
  4. Customise views, forms, and workflows in the editor
  5. Share the app link with your team — works in any browser or the AppSheet app
→ appsheet.com

Google Forms + Sheets automation
More powerful than most people realise — especially with automations
Fully included
▼

What to use it for

  • Run surveys that auto-populate a live Sheets dashboard
  • Conditional logic — show different questions based on answers
  • Auto-email a confirmation when someone submits a form
  • Use response validation to enforce data quality (e.g. date formats, numeric ranges)
  • Accept file uploads directly into Drive via form submissions

How to get started

  1. Create a form at forms.google.com
  2. Under "Responses" tab → click the Sheets icon to link a live spreadsheet
  3. In Forms settings, enable "Email notifications" for each response
  4. Add sections and conditional logic via the three-dot menu on each question
  5. In Sheets, use Apps Script or Workspace Studio to trigger follow-up actions
→ forms.google.com

Google Keep
Fast capture for notes, reminders, and lists — synced across devices
Fully included
▼

What to use it for

  • Capture meeting notes on your phone, sync instantly to desktop
  • Set location- or time-based reminders for follow-ups
  • Shared checklists with team members (e.g. event prep)
  • Save images with text — Keep extracts text from photos automatically
  • Pin a note to Google Docs via the Keep sidebar for reference while writing

How to get started

  1. Go to keep.google.com or install the mobile app
  2. Hit the "+" button to create a note, checklist, or voice recording
  3. Assign a colour and labels to organise notes by project
  4. Click the person icon on a note to share it with a colleague
  5. In Google Docs, open the Keep sidebar via Tools → Keep Notepad
→ keep.google.com

Google Sites
Build internal wikis, project pages, and resource hubs — no design skills needed
Fully included
▼

What to use it for

  • Build a team intranet with links to key docs, forms, and contacts
  • Create a project hub page — embed Sheets, Docs, Slides, and Meet links
  • Onboarding portal for new staff with resources and schedules
  • Program landing page for internal reporting or partner comms
  • Event page with agenda, speakers, and registration form embedded

How to get started

  1. Go to sites.google.com → click "+" to start a new site
  2. Pick a template or start blank — drag-and-drop layout editor
  3. Embed Google Docs, Sheets, Slides, Forms, Maps, or YouTube directly
  4. Set access: internal-only (Pratham team) or public link
  5. Publish and share the URL — no hosting setup required
→ sites.google.com

Looker Studio
Free dashboards and data visualisations from Sheets, Drive, or external sources
Free · separate from Workspace Data viz
▼

What to use it for

  • Turn a Sheets data dump into an interactive dashboard
  • Build a live program metrics report for leadership or donors
  • Visualise ASER or placement data with charts, maps, and filters
  • Connect multiple data sources into a single report
  • Share a live dashboard link — no login required for viewers

How to get started

  1. Go to lookerstudio.google.com — sign in with your Pratham account
  2. Click "Create → Report" and connect a data source (start with Google Sheets)
  3. Add charts, scorecards, and filters from the toolbar
  4. Use "Date range" controls to make the report interactive
  5. Click Share → get link to share with anyone (no account needed to view)
→ lookerstudio.google.com
